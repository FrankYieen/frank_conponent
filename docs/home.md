## 安装

```shell
$ npm i frank_component -S
```

或者

```shell
$ yarn add frank_component -S
```

## 样式引入

使用时如需要样式，请引入主样式文件 `frank_component/es/Style/index.css`;

例如 antdpro 中使用,只需在`global.less`文件头写入

```shell
@import '~frank_component/es/Style/index.css';
```
